1.资源下载git网址
https://github.com/topameng/protoc-gen-lua

2.下载安装python，配置环境变量

3.下载protobuf-2.5.0.rar，解压到c盘
C:\protobuf-2.5.0\python路径下进入cmd，执行python setup.py install

4.下载protoc-gen-lua-master.zip，解压到c盘
C:\protoc-gen-lua-master\example路径下放置要转换的proto文件，双击make_proto.bat，
就会在C:\protoc-gen-lua-master\example\Lua\Protol路径下生成对应的lua文件
